# Cloud Note 

CloudNote is a versatile note-taking application designed to streamline the process of capturing, organizing, and accessing your ideas, thoughts, and reminders across multiple devices.
With CloudNote, users can create, edit, and manage their notes effortlessly

### Project Name

- CloudNote

## Features

- **User-Friendly Interface:** The intuitive and easy-to-use interface allows you to create, edit, and manage your notes effortlessly.
The clean design ensures a distraction-free note-taking experience.

- **Rich Text Formatting:** Enhance your notes with rich text formatting options, including bold, italics, bullet points, and headings.
This helps in organizing your thoughts more clearly and making your notes more readable.

## Technologies Used

- **PHP:** Integration with backend APIs to fetch user's data, notes data, and other data.
- **MySQL:** Relational database management system for storing and retrieving data.
- **Android with Kotlin:** In Anroid Build the UI designing and In Kotlin Integration APIs
